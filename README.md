# Pannable

jQuery plugin to make an element pannable.

## Getting Started
Download the [production version][min] or the [development version][max].

[min]: https://raw.github.com/Andrea/pannable/master/dist/pannable.min.js
[max]: https://raw.github.com/Andrea/pannable/master/dist/pannable.js

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/pannable.min.js"></script>
<script>
jQuery(function($) {
  $.awesome(); // "awesome"
});
</script>
```
